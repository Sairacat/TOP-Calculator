Tips

1.This calculator only supports the calculation between two numbers

2.You can't enter a negative number manually. If you want to use a nagative number as operand, enter a postive one and press '+/-'

3.'/' stands for division, '*' stands for multiplication, '%' simply divides your input by 100 and returns the result.

4.You can change the operator you want BEFORE entering the second number
